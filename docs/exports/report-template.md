# Lead Intelligence Report Template

## Format Specification

This document defines the complete structure of the Lead Intelligence Report template. Every report generated by the platform follows this format exactly. The template is rendered as a PDF, with an identical markdown version available for copy-paste into Notion, Confluence, or Google Docs.
